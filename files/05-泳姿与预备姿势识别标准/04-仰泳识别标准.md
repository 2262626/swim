一、仰泳正确动作 & 姿势判断依据（教练级标准）
仰泳是仰卧在水面的泳姿，核心判断 5 个关键点：
1. 身体姿态
✅ 正确：身体保持平直、仰卧、核心收紧，髋部抬高不下沉，身体接近水平❌ 错误：腰部塌陷、臀部下沉、身体弯曲、整个人 “坐” 在水里
2. 手臂动作（最重要）
✅ 正确：
双臂交替轮流划水
出水时手臂完全伸直，肩膀带动出水
空中移臂保持直臂
入水时小指先入水，手臂伸直切入水中
❌ 错误：
手臂弯曲出水
双臂同时划水
拖肘、沉肘
动作不同步、乱划
3. 腿部动作
✅ 正确：
双腿交替上下打腿
膝盖微屈但不露出水面
由髋部发力，脚踝放松
❌ 错误：
膝盖弯曲过大、露出水面
双腿并拢不动
蹬腿像蛙泳
4. 身体滚动（转体）
✅ 正确：随手臂划水自然左右滚动（肩膀 + 髋部一起转动）❌ 错误：身体僵硬完全不转，只动手臂
5. 头部姿态
✅ 正确：头部稳定不动，眼睛看上方，不抬头、不低头、不晃❌ 错误：抬头看脚、低头看水、头部乱晃导致身体失衡

2. 仰泳判断必须计算的参数（开发直接用）
① 身体滚动（转体是否正确）
计算：左肩与右肩的 X 轴坐标差作用：判断身体是否左右滚动
② 手臂交替动作
计算：左手与右手的 Y 轴坐标差 & 运动时序作用：判断是否交替划水，而非同时划水
③ 手臂伸直度
计算：肩 → 肘 → 手 的夹角作用：判断是否直臂出水 / 移臂（正确≈180°）
④ 髋部是否下沉
计算：肩膀平均 Y 与 髋部平均 Y 的高度差作用：判断身体是否平直不下沉
⑤ 双腿交替打腿
计算：左右脚踝 Y 轴位移差作用：判断是否交替打腿
⑥ 膝盖弯曲度
计算：髋 → 膝 → 踝 的夹角作用：判断膝盖是否过度弯曲
⑦ 头部稳定性
计算：鼻子（头部）坐标波动幅度作用：判断头部是否稳定不乱晃

pose.onResults((results) => {
  const kp = results.poseLandmarks;
  if (!kp) return;

  // -------- 1. 身体滚动判断 --------
  const bodyRoll = Math.abs(kp[11].x - kp[12].x);
  if (bodyRoll < 0.03) {
    showTip("身体僵硬，需随划水自然滚动");
  }

  // -------- 2. 手臂交替划水判断 --------
  const handDiff = Math.abs(kp[15].y - kp[16].y);
  if (handDiff < 0.06) {
    showTip("请交替划臂，不要同时划水");
  }

  // -------- 3. 手臂是否伸直 --------
  const leftArmAngle = calculateAngle(kp[11], kp[13], kp[15]);
  if (leftArmAngle < 150) {
    showTip("手臂弯曲，仰泳需直臂出水");
  }

  // -------- 4. 髋部是否下沉 --------
  const shoulderAvgY = (kp[11].y + kp[12].y) / 2;
  const hipAvgY = (kp[23].y + kp[24].y) / 2;
  const hipDrop = shoulderAvgY - hipAvgY;
  if (hipDrop > 0.15) {
    showTip("髋部下沉，请收紧核心抬高身体");
  }

  // -------- 5. 双腿交替打腿 --------
  const ankleDiff = Math.abs(kp[27].y - kp[28].y);
  if (ankleDiff < 0.04) {
    showTip("请交替打腿");
  }

  // -------- 6. 头部稳定 --------
  const headMove = getHeadMovement(kp[0]);
  if (headMove > 0.02) {
    showTip("头部保持稳定，不要晃动");
  }
});
