一、蛙泳正确动作 & 姿势判断依据（教练级标准）
蛙泳是唯一手脚配合、节奏性极强的泳姿，核心看 5 个模块：
1. 手臂动作（对称循环）
✅ 正确：
双臂完全对称
划水：向外→向下→向内→胸前夹肘
前伸：双手并拢伸直向前滑行
❌ 错误：
双臂一快一慢、不对称
划水过宽、拖肘
不前伸、一直乱划
2. 腿部动作（最容易错）
✅ 正确：
收腿：慢收、膝盖不超过肩宽
翻脚：脚掌外翻对准水
蹬夹：向后蹬→快速并拢滑行
❌ 错误：
膝盖分得太开（比肩膀宽）
只蹬不夹、收腿过快
小腿乱踢、像自由泳打腿
3. 手腿配合（核心！）
✅ 正确：
手划 → 腿不动
收手 → 再收腿
伸手 → 再蹬腿
蹬完并拢滑行
❌ 错误：
手腿同时动
边划手边收腿
没有滑行阶段
4. 身体姿态
✅ 正确：
滑行时身体平直、紧绷、水平，髋部抬高
❌ 错误：
腰部塌陷、臀部下沉
身体起伏太大
5. 呼吸配合
✅ 正确：
手臂划水 / 夹肘时抬头吸气
手臂前伸时低头入水
❌ 错误：
抬头过高、身体抬起
一直抬头不低头

2. 必须计算的判断参数（开发直接复制）
① 双臂对称性
左右手 X/Y 坐标差
判断：是否同步、对称
② 手臂动作轨迹
双手之间距离（划水变宽 → 前伸变窄）
双手与肩膀的 Y 轴距离
判断：划水→夹肘→前伸
③ 膝盖宽度（关键纠错点）
双膝 X 轴距离
判断：不能 > 双肩宽度
④ 腿部动作角度
髋→膝→踝 夹角变化
判断：收腿→翻脚→蹬夹
⑤ 手腿动作时序
手动作阶段 vs 腿动作阶段
判断：手划腿不动、伸手再蹬腿
⑥ 身体水平度
肩膀 Y 与 髋部 Y 高度差
判断：髋部是否下沉
⑦ 呼吸时机
鼻子（头部）与肩膀高度差
判断：是否划水抬头、伸手低头

pose.onResults((results) => {
  const kp = results.poseLandmarks;
  if (!kp) return;

  // -------- 1. 双臂对称判断 --------
  const handDiffX = Math.abs(kp[15].x - kp[16].x);
  if (handDiffX > 0.1) {
    showTip("双臂不对称，请保持同步");
  }

  // -------- 2. 膝盖宽度判断（蛙泳核心） --------
  const kneeWidth = Math.abs(kp[25].x - kp[26].x);
  const shoulderWidth = Math.abs(kp[11].x - kp[12].x);
  if (kneeWidth > shoulderWidth) {
    showTip("膝盖太宽，不要超过肩宽");
  }

  // -------- 3. 手臂是否前伸 --------
  const handDistance = Math.abs(kp[15].x - kp[16].x);
  if (handDistance < 0.05) {
    // 双手并拢 = 前伸滑行状态
    showTip("前伸正确，请保持滑行");
  }

  // -------- 4. 髋部是否下沉 --------
  const shoulderY = (kp[11].y + kp[12].y) / 2;
  const hipY = (kp[23].y + kp[24].y) / 2;
  if (shoulderY - hipY > 0.15) {
    showTip("髋部下沉，请收紧核心");
  }

  // -------- 5. 呼吸：是否抬头 --------
  const headUp = kp[0].y < (kp[11].y + kp[12].y)/2 - 0.1;
  if (headUp) {
    showTip("抬头过高，身体不要抬起");
  }
});
