一、自由泳正确动作判断依据（教练级标准）
自由泳 = 爬泳，核心看 5 点：身体姿态、手臂动作、腿部动作、身体滚动、呼吸配合
1. 身体姿态（核心）
✅ 正确：身体保持水平、平直、紧绷，髋部抬高，减少阻力❌ 错误：髋部下坠、身体松软、腰部塌陷
2. 手臂动作（划水）
✅ 正确：
左右臂交替轮流划水
入水：手臂伸直，指尖先入水
划水：S 形划水，高肘抱水
出水：肩部带动，手臂自然放松出水
❌ 错误：
双臂同时划水
入水过深 / 过宽
沉肘、拖肘
手臂弯曲严重
3. 腿部动作（打腿）
✅ 正确：
双腿交替上下打腿
膝盖微屈（不露出水面）
脚踝放松，由髋部发力
❌ 错误：
双腿并拢不打腿
膝盖过度弯曲
蹬腿像蛙泳
水花过大但无推进力
4. 身体滚动（转体）
✅ 正确：绕身体纵轴左右滚动（肩 + 髋一起转），增加划水幅度❌ 错误：身体僵硬不转动，只动手臂
5. 呼吸配合
✅ 正确：身体滚动时转头吸气，头部稳定，不抬头、不抬肩❌ 错误：抬头呼吸、身体抬起、动作断裂

2. 必须计算的判断参数（开发直接用）
① 身体水平 / 直线度
肩、髋、踝的 Y 轴高度差
判断：髋部是否下沉
② 手臂交替划水
左手 vs 右手 的 Y 轴运动时序
判断：是否一上一下、交替运动
③ 手臂伸直度
肩 → 肘 → 手 的夹角
判定：接近 180°= 直臂（正确）
④ 身体滚动（转体）
左肩 vs 右肩 的 X 轴高度差
右髋 vs 左髋 的 X 轴高度差
判定：是否左右滚动
⑤ 双腿交替打腿
左膝 vs 右膝 的 Y 轴位移
左踝 vs 右踝 的 Y 轴位移
判定：是否交替上下
⑥ 膝盖弯曲度
髋 → 膝 → 踝 夹角
判定：不能过小（不能过度弯曲）
⑦ 呼吸动作
鼻子（头部）的左右转动幅度
头部与肩的相对位置
判定：是否转头呼吸，而非抬头

四、前端核心判断逻辑（伪代码，可直接用）

运行
pose.onResults((results) => {
  const kp = results.poseLandmarks;

  // 1. 身体滚动判断（肩左右差）
  const roll = Math.abs(kp[11].x - kp[12].x);
  if (roll < 0.03) tip("身体僵硬，请转体");

  // 2. 手臂交替判断（左右手Y差）
  const armAlternate = Math.abs(kp[15].y - kp[16].y);
  if (armAlternate < 0.05) tip("请交替划臂");

  // 3. 髋部是否下沉
  const hipDrop = (kp[11].y + kp[12].y)/2 - (kp[23].y + kp[24].y)/2;
  if (hipDrop > 0.15) tip("髋部下沉，请抬高身体");

  // 4. 双腿交替打腿
  const legAlternate = Math.abs(kp[27].y - kp[28].y);
  if (legAlternate < 0.04) tip("请交替打腿");

  // 5. 呼吸：是否抬头
  const headUp = kp[0].y < (kp[11].y + kp[12].y)/2 - 0.1;
  if (headUp) tip("不要抬头，请转头呼吸");
});
