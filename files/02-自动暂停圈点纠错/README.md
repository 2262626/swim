# 需求02：自动暂停 + 圈点勾画 + 纠错建议

## 目标

把视频分析升级为“可暂停讲解”的教学流程：在关键错误帧自动暂停，定位错误位置，给出可执行纠正建议。

## 当前状态

当前代码已经落地的核心能力是：

- 自动暂停
- 教学帧弹层
- 基于规则的错误提示与纠正建议
- 抓拍回放与会话上报

当前尚未完整落地的能力包括：

- 教练手动批注图层
- 轨迹线与错误编号气泡
- 独立报告页
- PDF 生成闭环

因此，本专题当前应按“教练辅助分析能力”理解，而不是“完整教学平台已全部实现”。

## 核心能力

- 自动暂停：关键错误、角度越界、时序异常、姿态丢失、阶段切换
- 当前圈点层：关键点、连线、角度、教学提示
- 状态分析：泳姿阶段、节奏、对称性、线型、呼吸配合
- 纠错建议：动作口令、目标区间、陆上/水中练习、复查标准

## 结果输出

- 当前结果：自动暂停事件、关键快照、Top问题与下次训练重点
- 可沉淀到成长记录的结构化结果
