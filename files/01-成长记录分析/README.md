# 需求01：游泳成长记录分析方案

## 目标

构建基于四泳姿评分（自由泳/蛙泳/蝶泳/仰泳）的个人成长分析体系，用于训练复盘、短板定位与阶段目标管理。

## 核心产出

- 单次训练分析：总分、分项分、阶段表现、关键角度、对称性
- 周/月趋势分析：均值、波动率、稳定区间、PR
- 训练负荷关联：训练量与质量关系
- 个人成长记录：周报、月报、阶段KPI

## 建议采集字段

- Session级：日期、时长、RPE、训练目标
- Stroke级：泳姿总分、分项分、划频、划臂计数
- Biomech级：关节角度、轴线偏移、时序差
- 备注级：教练点评、重点问题、训练作业
