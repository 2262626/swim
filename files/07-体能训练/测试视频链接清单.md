# 测试视频链接清单

更新时间：2026-03-31

本文只收录与当前首批已实现动作相对应的公开视频或带公开视频页面，方便你下载后做本地回放测试。

## 一、选片原则

建议优先下载满足以下条件的视频：

- 画质至少 `720p`
- 单动作持续 `15-30` 秒以上
- 镜头固定，不频繁剪辑
- 人物全身或关键关节完整入镜
- 器械路径清晰
- 水印、字幕、贴纸尽量少

不建议拿来做测试的视频：

- 镜头频繁切换
- 自拍镜头
- 只拍上半身或只拍腿
- 被器械、陪练或镜子严重遮挡
- 大量变速、慢动作、滤镜、拼接

## 三、推荐测试集组织方式

建议每个动作先收 `6-10` 条样片，按以下维度分桶：

- 机位：
  - 正面
  - 斜侧
  - 纯侧面
- 画质：
  - 高清
  - 中等
  - 较差
- 人群：
  - 青少年
  - 成年人
- 动作质量：
  - 标准动作
  - 常见错误动作

这样做的好处是：

- 你能快速看出哪类机位最稳定
- 能区分“算法问题”还是“视频条件问题”
- 后面做演示给老板时，证据会更完整
