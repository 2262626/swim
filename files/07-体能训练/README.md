# 07-体能训练

当前目录用于维护“固定动作体能训练姿势识别”专题文档，要求文档内容与当前前端真实能力保持一致。

## 当前文档

- `体能训练.md`
  - 模块目标、前端实现边界、当前已支持器械与二期范围说明
- `固定器械骨骼识别标准与提示词.md`
  - 首批支持动作的骨骼判定标准、阶段定义、评分项和双年龄段提示词策略
- `测试视频链接清单.md`
  - 2026-03-31 检索的公开视频/视频页面链接，可直接用于下载和回放测试
- `健身房各器械使用教学（单器械单独讲解文案）.md`
  - 用户提供的原始器械讲解素材

## 当前代码已落地能力

- 新增路由：`/fitness`
- 仅前端本地分析，不依赖后端
- 支持两种模式：
  - 实时分析
  - 本地视频分析
- 支持双年龄段语音提示：
  - 6-18 岁青少年
  - 18 岁以上成年人
- 支持开始按钮控制，不是进入页面就自动分析
- 支持按器械显示测试辅助线，优先跟随人物骨骼动态更新，并可在页面中手动开关
- 青少年模式保留单独语音提示，但不再额外显示顶部绿色提示条
- 页面布局按手机和 iPad 竖屏/横屏优先，不要求 PC 适配
- 首批已实现 5 个高确定性固定动作：
  - 史密斯深蹲
  - 坐姿腿屈伸
  - 坐姿划船
  - 龙门架夹胸
  - 罗马椅挺身

## 当前未实现能力

以下动作在原始体能素材中出现过，但当前版本没有接入识别逻辑，也不应在对外文档中描述为“已支持”：

- 上斜哑铃卧推
- 平板卧推
- 杠铃片取放流程
- 引体向上 / 双杠臂屈伸
- 坐姿腿弯举

## 维护原则

- 一期只保留“单机位、前端骨骼、确定性较高”的固定动作。
- 对于遮挡严重、卧姿/悬垂姿态、器械遮挡关键点过多的动作，统一标注为二期。
- 外链视频仅作为测试样片来源，不代表官方授权下载；使用前请自行确认版权与平台规则。
