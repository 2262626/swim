## 当前维护原则

- `files` 中的文档优先反映“当前代码真实能力”，避免把规划态能力写成已实现能力。

- 对于自动暂停、纠错建议、数据上报等主题，应优先参考各专题内最新的“当前方案/验证结论”文档。
