一、先明确：蝶泳核心正确动作判断依据（游泳标准）
蝶泳最关键的是躯干波浪、手臂动作、腿部鞭状打水、呼吸配合，错误姿势全部围绕这 4 点判断。
1. 躯干核心（波浪发力，最关键）
✅ 正确：腰腹带动全身波浪运动，髋部先下沉→上提，身体呈 S 形连贯波动
❌ 错误：只用手臂 / 腿发力、腰部僵硬、身体平直无波浪
2. 手臂动作
✅ 正确：双臂同步、对称、平直出水，入水时手臂前伸并拢
❌ 错误：双臂不同步、一高一低、出水弯曲、划水过宽 / 过窄
3. 腿部（鞭状打水）
✅ 正确：双腿并拢伸直，脚踝放松，由腰腹带动大腿→小腿→脚背鞭打
❌ 错误：双腿分开、膝盖过度弯曲、蹬腿像蛙泳、不同步
4. 呼吸配合
✅ 正确：手臂推水时抬头吸气，手臂前伸时低头入水
❌ 错误：抬头过早 / 过晚、身体抬太高（阻力大）

2. 蝶泳姿势判断 → 计算得到的核心参数
我直接给你开发用的公式 / 判断指标，不用你再推导：
① 双臂对称性（判断是否同步）
左右肩 Y 坐标差
左右肘 Y 坐标差
左右手 Y 坐标差
判定：差值 < 阈值 = 对称 / 正确，差值过大 = 不同步
② 手臂伸直度（判断出水是否笔直）
肩→肘→手 的夹角
判定：接近 180° = 手臂伸直（正确），角度小 = 手臂弯曲（错误）
③ 躯干波浪（核心指标）
肩膀与髋部的垂直距离差（Y 轴差）
肩高髋低 ↔ 髋高肩低 交替变化 = 有波浪（正确）
距离几乎不变 = 腰部僵硬（错误）
④ 腿部并拢度
左右膝、左右脚踝的水平距离（X 轴差）
判定：距离 < 阈值 = 并拢（正确），距离大 = 双腿分开（错误）
⑤ 腿部伸直度
髋→膝→踝 的夹角
判定：接近 180° = 伸直（正确），角度小 = 膝盖弯曲（错误）
⑥ 呼吸姿态
鼻子（头部）与肩膀的相对高度
推水时头部抬起 = 正确呼吸
